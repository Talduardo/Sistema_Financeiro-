# 💰 Sistema Financeiro em Python

Este projeto foi desenvolvido em **Python** com integração ao **Microsoft Excel**, com o objetivo de criar um sistema para **armazenar dados financeiros, relacionar pessoas e valores, gerar relatórios e visualizar resultados em tabelas e gráficos**.  
A aplicação permite acompanhar dívidas, calcular saldo final do dia e exportar dados em diferentes formatos.

---

## 🎯 Objetivos:

- Armazenar dados financeiros de forma estruturada.  
- Relacionar pessoas e valores (quem deve e quanto deve).  
- Criar listas de pessoas que estão devendo.  
- Calcular o **saldo final do dia**.  
- Gerar tabelas e gráficos para melhor visualização dos dados.  
- Exportar resultados em **CSV** e **Excel (.xlsx)**.  

---

## 📂 Estrutura do Projeto:

- **app_financeiro.py** → Script principal do sistema financeiro.  
- **figuras_geométricas.py** → Módulo adicional para cálculos geométricos.  
- **oficina5.py** → Exercícios e implementações complementares.  
- **requirements.txt** → Lista de dependências necessárias.  
- **README.md** → Documentação do projeto.  

---

## 🖥️ Tecnologias Utilizadas:

- **Python** → Linguagem principal para implementação.  
- **Microsoft Excel** → Exportação e visualização dos dados.  

### Bibliotecas utilizadas
- **Pandas** → Manipulação e análise de dados.  
- **NumPy** → Operações matemáticas.  
- **Matplotlib / Seaborn** → Criação de gráficos e visualizações.  
- **OpenPyXL** → Integração com arquivos Excel.
